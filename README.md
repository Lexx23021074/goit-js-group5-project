
# Wedding Photographer - MVP

Цей проєкт — це односторінковий сайт-візитка для весільного фотографа (Maria Kovalenko), розроблений у межах групової роботи на курсі FullStack.

## 📝 Про проєкт

Сайт допомагає фотографу презентувати свої послуги, портфоліо, відгуки клієнтів та збирати заявки на фотосесії через форму зворотного зв'язку.

## 🚀 Технології

* **HTML5** (семантична верстка)
* **CSS3** (Mobile First, Flexbox/Grid)
* **JavaScript** (ES6+ модулі, запити до API)
* **Vite** (збірник проєкту)
* **Swiper.js** (слайдер відгуків)
* **Axios** (робота з API)
* **Modern-normalize** (нормалізація стилів)

## 🛠 Як запустити проєкт

1. **Клонуйте репозиторій:**

```bash
git clone git@github.com:Lexx23021074/goit-js-group5-project.git

```

1. **Встановіть залежності:**

```bash
npm install

```

1. **Запустіть режим розробки:**

```bash
npm run dev

```

1. **Збірка проєкту для деплою:**

```bash
npm run build

```

## 🏗 Структура проєкту

Проєкт побудований за модульною структурою:

* `src/partials/` — HTML-частини кожної секції.
* `src/css/` — стилі для кожної секції (окремі файли).
* `src/js/` — логіка кожної секції (ES-модулі).
* `index.html` — головний файл, де збираються всі частини.
